# 공공API를 이용해 유기동물을 조회하는 서비스입니다!

## Description

🔎 현재 공공데이터포털에서 제공하는 유기동물 조회 오픈 API를 통해 유기동물을 조회한다.

🔑 OAuth 카카오 소셜 로그인을 통해 회원을 관리한다.

🧾 사용자가 자신의 최근 검색 기록을 볼 수 있다.
